<div>

Izmanto HTML dokumenta sadaļas definēšanai. Tags izveido konteineru HTML elementu skeletam, kuram vēlāk var piešķirt personību ar CSS vai JavaScript elementiem.

<span>

Izmanto atsevišķu iekšlīnijas elementu formatēšanai. Tā darbība ir salīdzināma ar <div> tagu, taču <span> tags konteineru veido kā daļu no jau esoša HTML elementa rindas.

<a>

Izmanto hiperlinku ievietošanai HTML dokumentā. Atkarībā no taga atribūta href pielietojuma linku var norādīt tā tiešajā formā vai izveidot tam nosaukumu.

mix-blend-mode

Šī CSS metode ļauj sapludināt vairākus dokumenta slāņus dažādos veidos. Piemērā dotajā mājaslapā fona bildēm mainās pa virsu uzpludinātā krāsa atkarībā no sadaļas.

shape-outside

Šī CSS metode ļauj aplauzt elementus apkārt citiem, piemēram, tekstu ap bildi. Aplaušanu var veikt noteiktām ģeometriskām formām vai arī brīvām formām, kā piemērā.
