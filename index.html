<div>

Izmanto HTML dokumenta sadaļas definēšanai. Tags izveido konteineri HTML elementu skeletam, kuram vēlāk var piešķirt personību ar CSS vai JavaScript elementiem.

<span>

Izmanto atsevišķu iekšlīnijas elementu formatēšanai. Tā darbība ir salīdzināma ar <div> tagu, taču <span> tags konteineri veido kā daļu no jau esoša HTML elementa rindas.

<a>

Izmanto hiperlinku ievietošanai HTML dokumentā. Atkarībā no taga atribūta href pielietojuma linku var norādīt tā tiešajā formā vai izveidot tam nosaukumu.


background-blend-mode

Šī CSS metode ļauj sapludināt vairākus dokumenta slāņus dažādos veidos. Piemērā dotajā mājaslapā fona bildēm mainās pa virsu uzpludinātā krāsa atkarībā no sadaļas.
